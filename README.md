# CarScan Monorepo

This repository is organized into two separate folders for easier navigation:

- `frontend/` - React + Vite application for image upload and car classification
- `backend/` - Flask backend and ML model server (currently empty placeholder)

## Frontend

Run the frontend from the `frontend` folder:

```bash
cd frontend
npm install
npm run dev
```

## Backend

The backend folder is ready for Flask or Python server code.

```bash
cd backend
# add your backend files here
```

## Notes

- `frontend/` contains the existing app code and Vite config.
- `backend/` should contain your Flask API, model wrapper, and related backend files.
